# Венеция — лендинг клиники зубной имплантации

Статический одностраничный сайт клиники **«Венеция»** (Москва): имплантация All-on-4 / All-on-6, протезирование «под ключ», запись на консультацию и расчёт стоимости.

Стек: **HTML5 + CSS3 + vanilla JS (ES modules)**. Без сборщиков и фреймворков.

---

## Содержание сайта

| Секция | Описание |
|--------|----------|
| **Шапка** | Логотип, телефон, «Перезвоните мне», sticky при скролле, бургер-меню |
| **Hero** | Видео с clip-path, карточка All-on-6, оффер, CTA и цена |
| **Результаты** | Приживаемость, карточки возражений, блок врача |
| **Под ключ** | Этапы имплантации и протезирования, цены, гарантии |
| **Акция / запись** | Таймер, форма телефона |
| **Врачи** | Сетка экспертов, «Показать больше» |
| **Консультация** | Форма + блоки акций (0 ₽) |
| **Отзывы** | Карусель карточек, рейтинги |
| **Калькулятор** | Квиз из 5 шагов → телефон (на странице и в модалке) |
| **Контакты** | Галерея, Яндекс.Карта, адреса |
| **Подвал** | Навигация, политика, обратный звонок |

Доп. страницы: `privacy.html`, `license.html`.

## Что реализовано

**UI / UX**
- Адаптив (десктоп / планшет / мобилка)
- Sticky-шапка, масштаб `zoom: 0.9` на ≤1680px
- Hover-эффекты по группам элементов
- Scroll-reveal, учёт `prefers-reduced-motion`

**Формы и модалки**
- Маска телефона, согласия на ПДн
- Модалки: обратный звонок и квиз-расчёт

**Доступность**
- Skip-link, landmarks, `aria-label` у ключевых контролов
- Клавиатурное управление видео и меню
- Осмысленные `alt`, подписи к полям телефона

**Производительность**
- Ленивая загрузка картинок и Яндекс.Карт
- Hero-видео не грузится на мобилке
- Шрифт Manrope через `<link>` + preconnect
